# Suzane.WebApp

## Beskrivelse
Suzane.WebApp er en ASP.NET Core MVC-applikasjon som inkluderer en startside og en søkeside. Startsiden viser en enkel melding, mens søkesiden lar brukere søke etter data og vise resultatene.

## Funksjoner
- **Startside**: Viser en velkomstmelding.
- **Søkeside**: Lar brukere søke etter data og viser resultatene i en ordnet liste.

## Installasjon
1. **Klon repoet**:
   

2. **Åpne prosjektet i Visual Studio**:
   - Åpne `Suzane.WebApp.sln` i Visual Studio.

3. **Installer avhengigheter**:
   - Visual Studio vil automatisk installere nødvendige NuGet-pakker.

## Bruk
1. **Start applikasjonen**:
   - Trykk `F5` eller `Ctrl+F5` i Visual Studio for å kjøre applikasjonen.

2. **Naviger til Startside**:
   - Åpne en nettleser og gå til `http://localhost:XXXX/`. Du vil se en velkomstmelding.

3. **Naviger til Søkeside**:
   - Åpne en nettleser og gå til `http://localhost:XXXX/SearchPage`. Du vil se et søkeskjema.
   - Skriv inn en søkestreng og trykk på "Søk"-knappen for å se resultatene.

## Avhengigheter
- .NET 6.0 eller nyere
- ASP.NET Core MVC
